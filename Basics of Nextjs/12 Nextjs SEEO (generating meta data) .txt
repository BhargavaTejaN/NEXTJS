
meta data is for changing the description and titile for each individual page 

we can achive it in 2 ways 

1) Static MetaData
2) Dynamic MetaData

Static MetaData

in page each we want to change we can include the below piece of code 

export const metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

Dynamic MetaData

export async function generateMetadata({ params }) {
  const post = await getData(params.id);

  return {
    title: post.title,
    description: post.desc,
  };
}
